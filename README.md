# Pipe Game Solver

## Introduction
An autonomous solver for N x N Pipe games using **Depth First Search (DFS)** algorithm. Developed as a personal project to practice AI logic.

## Features
- Custom game engine and collision logic.
- Pathfinding using DFS to find the correct pipe rotations.
- Performance monitoring (Time & RAM usage).

## How to run
1. Clone the repo.
2. Run `python main.py`.
